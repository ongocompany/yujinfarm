## 미해결 / 외부 의존 상태 ⚠️

이미지가 **아직 Readdy 외부 CDN에서 로드됨** — 구독 해지 시 깨질 수 있음.

JS chunk 내부에서 참조하는 외부 이미지 URL (총 21개):

- `public.readdy.ai/ai/img_res/*.jpg` — 11개 (AI 생성 이미지)
- `public.readdy.ai/gen_page/{readdy-logo,watermark}.png` — 2개 (Readdy 로고/워터마크 — 제거 검토)
- `static.readdy.ai/image/920c2b09bbc669bf604bbf774be46c1b/*` — 8개 (사용자 업로드 사진)

**로컬화 전략**: 21개 파일을 `public/assets/images/`로 다운 → JS chunk 파일에서 `sed`로 URL 일괄 치환 → minify된 JS라 치환이 깔끔하게 됨 (동일 URL 문자열이 그대로 박혀 있음).
